Artdrop
=======

Artdrop React App

# Setup

Clone the repo.
Run:
```bash
npm i
npm start
```

# Running Tests

Run the following two commands in separate terminals:
```bash
webpack --config webpack.test.config.js --watch
mocha test-build/tests.js  --watch
```
